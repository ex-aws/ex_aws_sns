## Installation

The package can be installed by adding `:ex_aws_sns` to your list of dependencies in `mix.exs`
along with `:ex_aws` and your preferred JSON codec / HTTP client:

```elixir
def deps do
  [
    {:ex_aws, "~> 2.0"},
    {:ex_aws_sns, "~> 2.0"},
    {:poison, "~> 3.0"},
    {:hackney, "~> 1.9"},
  ]
end
```
